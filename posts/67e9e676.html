  • next

    v8.2.1

前期准备

安装前的请具备以下条件 primary

  • npm-v7.5.4
  • node-v14.2.0
  • 创建一个站点文件夹
    mkdir myblog/

一、从 npm下载 hexo && next 主题

此处开始均在新建的 blog 目录下进行操作
$ cd myblog/

1.hexo安装

$ npm install -g hexo-cli $ npm init $ npm install $ hexo server

2.next安装

在 nexov8.0 后将 theme/next文件夹转移到了 node_modules/hexo-theme-next

$ npm install hexo-theme-next@latest
